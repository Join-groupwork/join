(async) greetings() → {Promise.<void>}

Displays a greeting message based on the current time of day and shows the name of the currently authenticated user. The function: - Determines the appropriate greeting ("Good morning", "Good afternoon", "Good evening") based on the user's local time. - Updates the greeting text in the DOM. - Checks whether a user is authenticated. - Displays the user's display name or email if logged in, otherwise falls back to "Guest". - Listens for authentication state changes and updates the name dynamically.

(async) todoTasks(tasks) → {Promise.<void>}

Counts the number of tasks with the status "todo" or "to do" and updates the corresponding value in the summary UI. The function also ensures backward compatibility by checking if a task uses the old `subtask` property instead of `status` and assigns it accordingly.
Parameters:
Name Type Description

Properties
Name Type Attributes Description

 The current status of the task (e.g. "todo", "in progress", "done").

 Legacy property used as a fallback for the task status.
