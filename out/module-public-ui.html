Module: public-ui

Public page bootstrapper: renders shared header/sidebar UI and wires up the header menu + logout. This module injects templates into the public pages (e.g. summary-guest/public area) and provides a logout button that signs out the current Firebase user. DOM requirements (IDs must exist in the HTML): - `header` : container for the header template - `sidebar` : container for the sidebar template - `headerMenue` : profile/menu button in the header - `headerMenueNav`: dropdown/navigation container toggled by profile button - `logoutBtn` : logout button External dependencies: - Template functions from `/public/js/public-templates.js` - Firebase Auth instance `auth` and `signOut()`
