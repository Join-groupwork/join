Module: public-templates

HTML template factory functions for public pages. Provides template helpers that return HTML strings for shared public UI parts: header, sidebar, and a signup success message.

Methods

(static) getHeaderTemplate() → {string}

Generates the header HTML for public pages. Includes: - Help icon (currently just a div, no link) - Profile button with a dropdown menu (Legal Notice / Privacy Policy / Logout) DOM hooks created by this template: - `#headerMenue` (toggle button) - `#headerMenueNav` (dropdown container) - `#logoutBtn` (logout trigger)
