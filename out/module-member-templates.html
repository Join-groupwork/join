Module: member-templates

HTML template factory functions for member pages. Provides functions that return HTML strings for commonly used UI parts: header, sidebar, add-task form, contact overlays, signup message, and board task cards.
Source:

Methods

(static) generateTodosHTML(id, title, category, description, priority) → {string}

Generates a task card HTML string for the board (drag-and-drop). NOTE: - `category` is used to build the CSS class name `task__category--${category}` - `priority` is used to build the icon path `/assets/icons/${priority}-prio-icon.svg`
Parameters:
Name Type Description
id string Task ID (used as DOM element `id` and for drag & drop).
title string Task title.
category Category Task category label.
description string Task description.
priority Priority Task priority label.
